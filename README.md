# Zelda-Like Game 🎮

A 2D action-adventure game inspired by classic Zelda. Explore a world filled with puzzles, enemies.

## Features 🚀

### Combat System ⚔️
- **Sword Combos**: Perform melee attacks with a sword combo system.

### Built With 🛠️
- **Engine**: [Godot Engine](https://godotengine.org/)
- **Language**: GDScript
